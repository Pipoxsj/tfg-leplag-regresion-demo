# Outputs

Esta carpeta contiene los resultados generados por el pipeline de entrenamiento:

| Archivo          | Descripción                                                    |
|------------------|----------------------------------------------------------------|
| `metrics.json`   | Métricas del modelo (MAE, RMSE, R²) en formato JSON           |
| `plot.png`       | Gráfico scatter: valores reales vs. predichos                  |

**Nota**: Los archivos se generan automáticamente al ejecutar `python src/train_model.py` o el notebook de Colab.
